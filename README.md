# easyros
ros的安装与开发工具

## 使用环境
Ubuntun 20.04.6 LTS  
ros仅限在该系统安装noetic版本  
## 使用指令
```
#安装指令（可选）
sudo apt install python3-pip

#换源指令（可选）
pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple

#安装指令
pip install easyros

#运行指令
easyros
```

## 功能
自动安装ros(包换源)  
github链接转镜像  
自动创造工作空间  


## 其它
因为ROS的环境安装过于复杂，所以我直接调用了鱼香的安装脚本 `https://github.com/fishros/install`
